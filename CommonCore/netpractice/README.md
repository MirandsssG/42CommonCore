# NetPractice


NetPractice is a networking project from the 42 School curriculum designed to teach the fundamentals of IP addressing and subnetting through interactive exercises.

The goal of the project is to configure networks correctly by assigning the right IP addresses and subnet masks so that devices can communicate with each other.


## Description

**NetPractice** is an interactive training tool where you solve networking problems by configuring different devices in a simulated network.

Each level presents a network diagram with multiple hosts, routers, switches and connections.
Your task is to correctly configure:

-IP addresses

-Subnet masks

If the configuration is correct, the network will be able to communicate successfully.

This project helps develop an understanding of:

-IPv4 addressing

-Subnetting

-Network segmentation

-Communication between hosts, routers and switches

-Broadcast addresses


---


## Instructions


To run the project, type:
```bash
 ./run.sh
```

In the window that pops up we can start in either **Training mode** or **Evaluation mode**, choose the most adequate one.

After running any of the tests, we can click **Check again** to check if the IP's and Mask's that we inputted are correct, if they are, a button saying **Next Level** will pop up so that we can move on to the next level.

Whenever we want to, we can click the **Get my config** button and it will extract a .json file with our current configuration.
